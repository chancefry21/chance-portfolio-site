March Madness Predictive Modeling

Tools: Python, Pandas, Scikit-Learn, Matplotlib

Skills Demonstrated: Data cleaning, merging multiple datasets, feature engineering, model building, evaluation, sports analytics

Built a logistic regression model to predict NCAA March Madness matchups using team strength indicators from multiple datasets. After merging FiveThirtyEight ratings, historical matchups, and HeatCheck analytics, I engineered features such as power rating difference, seed difference, and pool value difference to help explain game outcomes.

Key Results:

  • Training Accuracy: 72%
  • Testing Accuracy: 63%
  • Power rating difference showed a strong relationship with actual score results
  • Model identified real matchup patterns despite the unpredictability of March Madness
